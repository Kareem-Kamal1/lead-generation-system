# Lead Generation System 🚀

## Overview
An automated lead generation system that captures leads from a web form, analyzes them with AI, and stores them in Google Sheets while sending email notifications.

## Features
- HTML form for lead capture
- AI-powered lead analysis using Groq LLaMA
- Automated email notifications via Gmail
- Lead storage in Google Sheets

## Tech Stack
- n8n (Automation)
- Groq AI (LLaMA model)
- Google Sheets
- Gmail

## Workflow
1. User fills out the HTML form
2. Webhook receives the lead data
3. Groq AI analyzes the lead
4. Lead stored in Google Sheets
5. Email notification sent via Gmail
